\section{Многокритериальная оптимизация.}

Добавляем в задачу несколько критериев оптимизации.

\D{
    Доминирование по Парето:

    $P \prec Q \leftrightarrow \forall i: P_i \leq Q_i
    \exists j: P_j < Q_j$

    Оптимальность по Парето = состояние системы, в котором ни
    один из показателей не может быть улучшен без ухудшения
    другого.
}

\D{
    \textbf{Множество Парето} = множество всех решений,
    оптимальных по Парето.

    \textbf{Фронт Парето} = множество всех критериев таких
    решений.

    \textbf{Текущее приближение} = лучшие решения из имеющихся
    в данный момент.
}
